## Visão geral

A infraestrutura possui dois ambientes VMware distintos:

1. **Appliance VMware dedicado ao OPNsense**
2. **Servidor VMware principal**, responsável pelas máquinas virtuais internas

## VMware 1 — Appliance do firewall

| Item | Informação |
| --- | --- |
| Função | Hospedar apenas a máquina virtual do OPNsense |
| Observação | Ambiente dedicado ao firewall |
| Rede | VLAN 1 |
| IP de gestão | 192.168.2.3 |

## VMware 2 — Servidor principal

| Item | Informação |
| --- | --- |
| Função | Hospedar os serviços internos da empresa |
| IP do host | `192.168.100.253` |
| Usuário padrão | `root` |
| Rede principal | VLAN 100 |
| Ambiente | Servidor VMware principal |

  A senha do VMware deve ser consultada no Passbolt. Não registrar senha em texto aberto nesta documentação.
</Warning>

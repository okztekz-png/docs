## Possíveis dependências

O DNS pode impactar:

- acesso ao Nextcloud;
- acesso a compartilhamentos do UCS;
- resolução de nomes internos;
- dispositivos UniFi;
- acesso a sistemas internos;
- navegação dos clientes DHCP.

## Pontos a confirmar

- O OPNsense entrega DNS próprio ou DNS público via DHCP?
- O UCS atua como servidor DNS interno?
- Existe domínio local no Univention?
- Existe domínio público relacionado ao Nextcloud?
- Existe Cloudflare ou outro provedor DNS externo?
- Quais DNS são entregues para VLAN 1, VLAN 10 e VLAN 100?

## Serviços críticos

### Firewall OPNsense

O OPNsense é crítico porque concentra:

- internet;
- failover;
- roteamento entre VLANs;
- DHCP;
- gateway das redes internas.

### UCS / Nextcloud

O UCS é crítico porque hospeda:

- serviços de arquivos;
- Nextcloud;
- possíveis serviços internos relacionados.

### UniFi Controller

O UniFi Controller é importante para:

- gerenciamento do AP;
- SSIDs;
- configurações de Wi-Fi;
- VLANs aplicadas às redes sem fio.

## Sugestão de checks

- Ping no firewall `192.168.2.1`.
- Ping no switch `192.168.100.2`.
- Ping no VMware Host `192.168.100.253`.
- Ping no UCS `192.168.100.254`.
- Ping no UniFi Controller `192.168.100.194`.
- Monitoramento de WAN 1 / Infortel.
- Monitoramento de WAN 2 / Starlink.
- Verificação do failover.
- Monitoramento de espaço em disco do UCS.
- Monitoramento de datastore do VMware Host.
- Monitoramento dos APs UniFi no controller.

## Riscos observados

<Warning>
  Ainda não há documentação suficiente sobre backup e restauração do ambiente. Essa informação deve ser levantada com prioridade.
</Warning>

Possíveis riscos:

- falha do VMware Host principal;
- perda de dados do UCS;
- perda de arquivos do Nextcloud;
- perda da configuração do OPNsense;
- perda da configuração do UniFi Controller;
- indisponibilidade da rede após falha de firewall;
- ausência de cópia externa;
- ausência de teste de restauração.

## Prioridades de backup

1. Configuração do OPNsense.
2. UCS / arquivos / Nextcloud.
3. UniFi Controller.
4. VM Windows.
5. Configuração do switch TP-Link.
6. Documentação de restauração do ambiente VMware.

## Pontos a confirmar

- Existe backup atualmente?
- O backup é feito pelo VMware, pelo sistema operacional ou por ferramenta externa?
- Existe backup externo?
- Existe retenção diária, semanal ou mensal?
- Onde ficam armazenados os backups?
- Quem verifica os backups?
- Já foi feito teste de restauração?
- Existe exportação periódica da configuração do OPNsense?
- Existe exportação da configuração UniFi?

## Acessos externos conhecidos

### SIAF via Winflector

O sistema **SIAF** possui acesso externo através do **Winflector**, que roda no servidor `192.168.1.253`.

Esse acesso é utilizado pelo notebook do **Sr. Jodano**.

## Dependências

- Servidor `192.168.1.253` online.
- Winflector em execução.
- Encaminhamento de portas ativo no roteador.
- Link de internet funcionando.
- Regras de firewall/NAT preservadas.

## Status atual

Até o momento, não há VPN documentada para o ambiente da **Jodano**.

## Acesso remoto conhecido

O acesso remoto conhecido atualmente é feito através do **Winflector**, utilizado para acesso externo ao sistema **SIAF**.

| Serviço | Tipo | Servidor | Observação |
| --- | --- | --- | --- |
| Winflector / SIAF | Acesso remoto de aplicação | `192.168.1.253` | Usado pelo notebook do Sr. Jodano |

## Pontos a confirmar

- Se existe alguma VPN configurada no roteador.
- Se existe VPN para acesso técnico.
- Se existe VPN para acesso do cliente.
- Se o acesso externo ao SIAF depende exclusivamente do Winflector.
- Se há usuários remotos adicionais além do notebook do Sr. Jodano.

## Observações

<Note>
  Caso seja implementada uma VPN futuramente, documentar tipo, usuários autorizados, rotas, portas, autenticação e procedimento de conexão.
</Note>

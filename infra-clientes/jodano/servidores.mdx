- Proxmox como host de virtualização.

## Pontos a confirmar

- Não há outras VMs além do servidor principal.
- Recursos da VM Servidor: CPU 6 Nucleos, 12 GB RAM e 256GB de SSD.
- Não há políticas de backup da VM, somente é feito snapshots quando vai atualizar o Lazzumix.
- Firewall interno do Windows é gerenciado pelo Acronis (gerenciado pela Tekz).
